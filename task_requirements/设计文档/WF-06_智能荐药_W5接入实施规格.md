## 2. Dify 控制台编排步骤（1.11.2）

1. 新建 **Workflow App**，命名建议：`WF-06 智能荐药`。
2. 按设计文档 **第 5 节** 逐节点创建（10 节点 + 双 End）。
3. HTTP 节点 5 配置：
   - URL: `http://172.17.0.1:8098/api/physician/internal/drugs/ai-search`（Docker 内访问 **ai-catalog-service**）
   - Header: `Authorization: Bearer <INTERNAL_AI_TOKEN>`
   - Body: `{{构造药品检索请求.search_request_body}}`
4. 发布工作流，复制 **API Key**（`app-xxx`）到 `.env`：

```bash
DIFY_WORKFLOW_W5=true
DIFY_API_KEY_W5=app-xxxxxxxx
```

5. 重启 `physician-service`，日志应出现：`W5 智能荐药：已启用 Dify Workflow`。
